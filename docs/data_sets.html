Public Data

Other sources of public fisheries data sets are:

  • RAM Legacy
  • Global Population Dynamics Database
  • NOAA NWFSC Salmon Population Summary
  • FAO Catch Databases
  • Ecomarres Global Catch database; paper on the Global Catch Database
  • Sea Around Us databases
  • Global Fishing Watch data
  • fishR databases
  • Salmon fecundity trends in the PNW github repo. This is a large dataset provided in this 2023 paper.
  • FISHGLOB global trawl survey database 2023
  • Moored bouy data from the Bering Sea
  • California Current survey data oceanographic and biological
  • PNW Zooplankton data
  • EFI RCN NEON Ecological Forecast Challenge data sets. Not fisheries. All the data are provided and the challenge lays out the goals (what to forecast) for each challenge. The Aquatic Ecosystems, Tick Abundance, and Beetle Abundance challenges would be appropriate for the class.
  • Greg Ruggerone and Jim Irvine, entitled “Numbers and biomass of natural- and hatchery-origin Pink, Chum, and Sockeye Salmon in the North Pacific Ocean, 1925-2015” in Marine and Coastal Fisheries: Dynamics, Management, and Ecosystem Science 10. DOI: 10.1002/mcf2.10023. Excel file of the data

Marine environmental data. Sound ideas

  • https://ecowatch.noaa.gov/ This doesn’t have the data but will point you to the data and is a vetted source of good marine indicators
  • https://coastwatch.pfeg.noaa.gov/EDC/index.html This is a tool that can help you get remote-sensing data

Climate data hubs

  • https://www.climatehubs.usda.gov/hubs/northwest.
  • https://drought.climate.umt.edu/
  • https://wrcc.dri.edu/wwdt/ West Wide Drought Tracker.
  • https://climatetoolbox.org/ Links to a bunch of climate dashboards
  • https://waterwatch.usgs.gov/ Simple dashboard with water products. Streamflow one is here.
  • https://www.nwrfc.noaa.gov/rfc/ NW River Forecast
  • http://ncdc.noaa.gov/temp-and-precip Links to a bunch of products/tools for drought
  • https://www.nwrfc.noaa.gov/water_supply/wy_summary/wy_summary.php?tab=2. Precip forecasts.
  • https://www.nwrfc.noaa.gov/snow/ Snow level status
  • https://www.nwrfc.noaa.gov/ws/ Water supply forecasts
  • https://www.nwrfc.noaa.gov/natural/plot/nat_forecasts.php?id=HWRI1 Water supply forecast for a specific location
  • https://www.cpc.ncep.noaa.gov/products/Soilmst_Monitoring/US/Soilmst/Soilmst.shtml Soil moisture products

Movement data

  • Polar bear argos and GPS data
  • High arctic argos and GPS data
  • Fin whale tracks
  • Dryad.org Search for “argos” for more track data
  • rerddap vignette Vignette showing you how to plot and get marine data for locations of a movement track using the rerddap R package
